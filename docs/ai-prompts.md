# AI Prompts

Ниже промты для генерации дополнительных ассетов в том же визуальном языке, что и приложение.

## Общие требования

- Формат: `PNG`, без текста, без эмодзи.
- Стиль: premium mobile product, dark aquatic luxury, glassmorphism-compatible.
- Палитра: deep navy, petrol blue, emerald, cold silver, soft warm gold accents.
- Запрещено: purple palette, cartoon style, childish gradients, glossy skeuomorphism.

## 1. Hero Background для главного экрана

```text
Create a premium mobile app hero background as high-resolution PNG.
Style: dark aquatic luxury, modern fintech-grade polish, subtle depth, atmospheric but clean.
Scene: abstract river surface seen from above, soft contour waves, layered translucent currents, faint sonar rings, elegant gradient fog.
Colors: deep navy, midnight blue, petrol, emerald highlights, cold silver reflections.
Composition: centered for portrait mobile screen, safe for UI overlays, no text, no icons, no people.
Output: one clean high-resolution PNG, portrait-friendly, high contrast for white foreground text.
```

## 2. Набор фоновых карточек для водоемов

```text
Generate a set of 4 premium location card backgrounds as PNG for a fishing mobile app.
Themes:
1. wide river backwater at dawn
2. clear lake with pine silhouettes
3. paid carp pond with geometric piers
4. private northern club lake with mountain-like shore lines
Style: semi-abstract vector illustration, cinematic, minimal detail, expensive look.
No text, no people, no fish photos, no emoji.
Colors: dark blue, slate, teal, muted amber, icy cyan.
Each background should work behind UI cards with readable overlay text.
```

## 3. Премиальные empty-state иллюстрации

```text
Create 3 premium empty-state illustrations in PNG for a dark fishing social app.
Subjects:
1. empty chat with sonar ripple motif
2. no catches yet with elegant lure silhouette
3. no saved places with map pin and contour waves
Style: thin vector lines, subtle glassmorphism highlights, modern product illustration, restrained and expensive.
No mascots, no emoji, no text.
Use a dark background-friendly palette with emerald and ice-blue accents.
```

## 4. Бейджи для видов рыбы

```text
Generate a pack of premium circular species badges as PNG with transparent background.
Species: pike, zander, perch, carp, trout, catfish.
Style: minimalist line icon inside layered ring, not cartoon, not realistic, suitable for dark UI chips.
Colors: soft emerald, cold silver, pale gold, deep navy background support.
No text, no gradients that look cheap, no emoji look.
Output each badge on transparent background.
```

## 5. Иконка приложения / splash mark

```text
Design a premium app mark as PNG for a fishing social network.
Concept: combine a map pin, ripple wave, and fish-hook geometry into one clean symbol.
Style: minimal, bold, geometric, premium startup brand, suitable for app icon and splash screen.
No text, no emoji, no mascot.
Palette: dark navy base with emerald highlight and optional silver secondary stroke.
Output on transparent background plus a square version on dark background.
```

## 6. Декоративный паттерн для чатов и фонов

```text
Create a seamless PNG pattern for a premium dark mobile chat background.
Motif: sonar rings, contour lines, water current curves, tiny geometric spark points.
Look: subtle, elegant, low-contrast, luxurious, not busy.
Colors: charcoal navy, petrol, muted cyan, soft emerald.
No text, no icons, no emoji, no obvious grid.
Designed to sit behind glass cards without reducing readability.
```
