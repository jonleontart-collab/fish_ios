# Modal Background Prompts

Use these prompts to generate dark, premium image backdrops for modal windows.

Recommended output folder:

- `public/images/modal-backgrounds/`

Recommended file names:

- `public/images/modal-backgrounds/map-discovery-bg.png`
- `public/images/modal-backgrounds/chat-sheet-bg.png`
- `public/images/modal-backgrounds/profile-panel-bg.png`
- `public/images/modal-backgrounds/notification-center-bg.png`

Purpose:

- `map-discovery-bg.png`: drawers and map search panels
- `chat-sheet-bg.png`: create-chat, join-chat, message action sheets
- `profile-panel-bg.png`: profile edit and public profile overlays
- `notification-center-bg.png`: notification drawer and compact system alerts
